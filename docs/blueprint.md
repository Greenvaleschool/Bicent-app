# **App Name**: Bicent

## Core Features:

- My Pal AI Assistant: An AI tool that provides natural-language chat, personalized trade ideas (entry, stop, target), and trend summaries based on real-time sentiment and technical data.
- Real-time Trading Terminal: Interactive charting interface with 1m-1d candle views and low-latency WebSocket feeds for sub-500ms market updates across stocks, gold, and crypto.
- Multi-Asset Portfolio Analytics: Aggregated dashboard tracking realized and unrealized gains across diverse asset classes with historical performance visualization.
- Automated Trend Detection: Algorithmic engine that classifies multi-timeframe trends using moving-average crosses, MACD, and RSI divergence signals.
- Smart Alert System: Configurable notifications for breakout/breakdown patterns, news sentiment shifts, and unusual volume spikes.
- Paper Trading Simulator: A risk-free environment utilizing simulated liquidity and slippage settings to test strategies prior to live execution.
- Predictive Risk Modeling: An AI tool designed to execute pre-trade safety checks, margin impact analysis, and confidence scoring for every trade recommendation.
- Secure Compliance Gateway: Robust security framework featuring 2FA (TOTP/WebAuthn), encryption-at-rest, and progressive KYC hooks for fiat connectivity.

## Style Guidelines:

- Background: Deep Obsidian (#0A0B0E) for an elite, high-focus dark mode environment.
- Primary: Vibrant Azure (#2563EB) and Accent: Electric Indigo (#8B5CF6) for critical trade actions and signals.
- Headlines use 'Space Grotesk' for a modern, computerized terminal aesthetic.
- Body copy uses 'Inter' for maximum legibility of dense numerical data and text summaries.
- Monospaced data uses 'Source Code Pro' specifically for order payloads, wallet addresses, and technical parameters.
- Multi-pane dashboard layout allowing synchronized viewing of charts, order entry, and the My Pal chat interface.
- Subtle flashing state changes for real-time price ticks and smooth transitions for modal order confirmations.
